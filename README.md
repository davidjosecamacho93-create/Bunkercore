# Bunkercore: Identidad Zero-Trust Universal

Bunkercore es el motor de identidad y operaciones de nueva generación. Basado en **FIDO2/WebAuthn**, elimina las contraseñas tradicionales para garantizar que el autor legítimo sea siempre quien interactúa con el sistema.

## Características
- **Identidad Criptográfica:** Registro biométrico permanente en Neon.
- **Arquitectura Multi-Tenant:** Soporte universal para múltiples negocios.
- **Operaciones Seguras:** Gestión de inventarios, cotización en tiempo real y control de merma.
- **Control de Acceso (RBAC):** Roles definidos por puesto para seguridad total.

## Tecnologías
- Node.js / Express
- PostgreSQL (Neon)
- WebAuthn / FIDO2
